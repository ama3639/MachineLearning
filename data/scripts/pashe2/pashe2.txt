## 🧪 تست سیستم:

### **تست 1: ثبت نام کاربر**
```
1. برو به: http://localhost:8000/register
2. نام کاربری: test1
3. رمز عبور: 123456
4. تلگرام ID: شناسه خودت
5. ثبت نام ← ورود
```

### **تست 2: دریافت سیگنال**
```
1. در داشبورد: BTC/USDT انتخاب کن
2. قیمت فعلی: 45000
3. حجم: 1000  
4. دکمه "دریافت سیگنال" ← نتیجه نمایش داده می‌شود
```

### **تست 3: API Authentication**
```bash
# تست مستقیم API
curl -X POST http://localhost:5000/predict \
  -u "test1:123456" \
  -H "Content-Type: application/json" \
  -d '{"close": 45000, "volume": 1000, "rsi": 50}'
```
